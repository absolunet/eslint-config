@absolunet/eslint-config-pwastudio

ESLint config for PWA Studio

Opinionated ESLint shareable config for PWA Studio

Extends eslint:recommended, jsx-a11y/recommended, package-json/recommended and @absolunet/eslint-config-react

Install

$ npm install @absolunet/eslint-config-pwastudio

Usage

Add some ESLint config to your .eslintrc.yaml:

extends: '@absolunet/pwastudio'

Used plugins

  • eslint-plugin-package-json - package.json files rules
