@absolunet/eslint-config-node-package

ESLint config for Node.js packages

Opinionated ESLint shareable config for Node.js packages

Extends @absolunet/eslint-config-node and @absolunet/eslint-config-package

Install

$ npm install @absolunet/eslint-config-node-package

Usage

Add some ESLint config to your .eslintrc.yaml:

extends: '@absolunet/node-package'

Used plugins

  • eslint-plugin-import - import/export syntax rules
