@absolunet/eslint-config-package

ESLint config for packages

Opinionated ESLint shareable config for packages

Extends @absolunet/eslint-config-jsdoc

Made to be combined with a @absolunet/eslint-config-core sourced config.

Install

$ npm install @absolunet/eslint-config-package

Usage

Add some ESLint config to your .eslintrc.yaml:

extends: '@absolunet/package'
